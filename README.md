Sistema Especialista – Diagnóstico de Doenças em Videiras

Alunas: Allana Paganini e Bianca Maccarini

Descrição: Sistema especialista que identifica doenças em videiras a partir de sintomas visuais. Utiliza regras de produção e encadeamento para frente com Experta e interface web via Flask.

Sintomas Monitorados:
Manchas amarelas, Pó branco, Frutos escurecidos, Folhas murchas, Lesões no caule, Manchas marrons, Mofo cinza, Galhos quebradiços, Presença de insetos, Solo encharcado

Doenças/Problemas Identificados:
Míldio, Oídio, Podridão Negra, Cancro, Botrytis, Estresse Hídrico, Infestação de Ácaros, Asfixia Radicular, Infecção Combinada (Míldio + Oídio), Podridão Ácida

Tecnologias:
Python 3.10.11, Flask, Experta, HTML/CSS

Instruções para rodar o projeto após clonar o repositório:

Abra o terminal do vscode e digite os seguintes comandos:
1. python -m venv venv
2. Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
3. venv\Scripts\activate
4. python app.py
